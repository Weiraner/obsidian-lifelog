# Lifelog — Dashboard

Open this note in **Reading view** (or Live Preview) with the Lifelog plugin
enabled. Both panels read the generated demo data under `.lifelog/daily/`.

## 时间块日历 / Time-block calendar

```lifelog-calendar
```

## 记账 / Expenses

```lifelog-expense
```

## 时间统计 / Time stats

```lifelog-stats
```

---

*Override a data source per block with a `key: value` line, e.g.:*

```lifelog-expense
dataRoot: .lifelog/daily
```
